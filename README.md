This program offers a modern, GUI-based banking simulator built in Java. This application provides a high-contrast and extremely user-friendly interface for managing account balances with real-time transaction tracking and secure session handling.

---

## ✨ Program Features
*   **Modern Dark Mode UI:** This code has been designed with a custom color palette for a professional look and feel.
*   **Live Dashboard:** Displays current available balance with localized currency formatting ($ USD).
*   **Transaction Log:** Real-time activity feed that tracks deposits and withdrawals during your session.
*   **Robust Error Handling:** Validates numeric inputs and prevents overdrafts with automated alerts.
*   **Session Summary:** Prompts for an opening balance and provides a final receipt upon closing.

## 🖥️ Usage Instructions
1.  **Opening Balance:** On launch of the application, enter your starting amount.
2.  **Transactions:** Enter a positive amount you would like to deposit or withdraw in the input field.
3.  **Deposit/Withdraw:** Use the color-coded buttons to update your balance.
4.  **Confirmations:** Follow the pop-up prompts to continue or end your session.
5.  **Receipt:** View your final balance in the exit dialog before the program closes out.
